HTML
HyperText Markup Language — the structure of web pages.
CSS
Cascading Style Sheets — controls the presentation and layout.
JavaScript
A programming language that adds interactivity to web pages.
DOM
Document Object Model — the browser's tree representation of HTML that JavaScript can manipulate.

Image with alt, width, height

Always include alt text for accessibility and width/height to prevent layout shift.

Figure with Figcaption

<figure> groups an image with its caption. Screen readers associate the caption with the image.
