# JavaScript

Practice and reference for the most commonly used JavaScript concepts — each file describes the problem and shows the working solution.

| # | File | Concepts Covered |
|---|------|-----------------|
| 1 | [String Methods](01-string-methods.js) | `split`, `join`, `slice`, `trim`, `replace`, `includes`, template literals, `padStart` |
| 2 | [Destructuring](02-destructuring.js) | Array & object destructuring, skip, rest, rename, defaults, nested, function params, swap |
| 3 | [Array Methods](03-array-methods.js) | `map`, `filter`, `find`, `reduce`, `some`/`every`, `sort`, `flat`, `flatMap`, chaining |
| 4 | [DOM Manipulation](04-dom-manipulation.html) | `querySelector`, `createElement`, `classList`, `appendChild`, mini todo list |
| 5 | [Event Handling](05-event-handling.html) | Click, keyboard, bubbling, `stopPropagation`, delegation, `preventDefault` |
| 6 | [Async JS](06-async-js.js) | Promises, chaining, `async`/`await`, `try`/`catch`, `Promise.all`/`race`/`allSettled` |
| 7 | [ES6+ Features](07-es6-features.js) | Spread, rest, `?.`, `??`, `Map`, `Set`, `for...of`/`for...in`, tagged templates |
| 8 | [Objects & Classes](08-objects-classes.js) | `Object.keys`/`values`/`entries`, classes, inheritance, `static`, private `#fields` |

**`.js` files** — Run with `node filename.js`
**`.html` files** — Open in browser, see the interactive result + check DevTools console
