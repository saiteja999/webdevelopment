When to Use Which Unit

This box uses: width in %, max-width in rem, padding in em, font-size in rem, border-radius in px, min-height in vh.

% — relative to parent (good for widths)

rem — relative to root font-size, 1rem = 16px default (consistent spacing)

em — relative to the element's own font-size (component-scoped)

vh/vw — relative to viewport height/width (hero sections, full-screen layouts)

px — absolute pixels (borders, shadows, small fixed details)
