CSS Box Model

box-sizing: border-box

Both boxes have width: 200px but different padding and border. With box-sizing: border-box, the 200px includes everything, so they render at the same total width.

Box A: 20px padding, 5px border
Box B: 10px padding, 10px border

Margin Collapse

Normally, adjacent vertical margins collapse — the gap between these would be 30px (not 60px). Using display: flex on the parent prevents collapse, so both margins apply and the gap becomes 60px.

margin-bottom: 30px

margin-top: 30px

Centering with Flexbox

The classic centering pattern: display: flex + justify-content: center (horizontal) + align-items: center (vertical) on the parent.

Centered!

Outline vs Border

border affects layout (takes up space). outline does not — it draws on top without shifting anything. outline-offset controls the gap between outline and border.
